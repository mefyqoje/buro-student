# Бюро студенческих работ

Статический сайт для организации "Бюро студенческих работ". Его можно открыть локально как обычный HTML-файл или
опубликовать через GitHub Pages.

## Файлы

- `index.html` — структура страницы и контент.
- `styles.css` — адаптивная визуальная система.
- `script.js` — мобильное меню, год в футере и подготовка письма из формы.
- `assets/hero-workspace.png` — hero-изображение для первого экрана.
- `assets/logo-mark.png` — логотип в шапке сайта.
- `assets/logo-icon.png` — иконка вкладки браузера.

## Публикация через GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите файлы проекта в корень репозитория.
3. Откройте `Settings` → `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/root`, затем сохраните.

Перед публикацией замените контакты `info@buro-student.ru` и `+7 000 000-00-00` на реальные.
